## Frontend Design & Components

Use this skill when working on CodeCast's frontend design, UI components, styling, or layout improvements.

### What you can do:
1. **Review component structure** - Analyze existing components in `/client`
2. **Design new components** - Create component specs and implementations
3. **Improve styling** - Update CSS/styling for better UX
4. **Check component hierarchy** - Verify component organization and reusability
5. **Run dev server** - Start the client in dev mode for testing
6. **Review design consistency** - Ensure UI consistency across components

### Key directories:
- `/client` - Frontend source code
- `/client/src/components` - React components
- `/client/src/styles` - Stylesheets

### Common tasks:
- Creating new component files
- Updating component props and behavior
- Refactoring styles for better maintainability
- Testing component changes in browser
- Ensuring responsive design

When working on frontend design, verify changes in the running dev server before finalizing.
